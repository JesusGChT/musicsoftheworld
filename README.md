# Musics of the World 🌍🎵

**A musical journey around the world.**

Discover traditional music, instruments and rhythms from every corner of the planet. An interactive atlas for travellers, curious minds, students and music lovers.

## 🗺️ What is this?

Musics of the World is an interactive map that connects geography with traditional music. Click on any region to discover:

- The main musical genres of that place
- The characteristic instruments
- Videos and audio samples
- Cultural context for travellers

## 📁 Structure

```
musicsoftheworld/

├── css/
│   └── style.css       # All styles
├── js/
│   └── app.js          # Map and panel logic
└── data/
    └── regions.json    # All musical content (edit this to add regions)
```

## ✏️ Adding a new region

To add a new region, edit `data/regions.json` and add a new object to the `regions` array following the existing structure.
